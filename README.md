## Tic-Tac-Toe (Unity)

This repository contains a Unity (2022.3 LTS) Tic-Tac-Toe mini-game for local multiplayer (2 players on the same device), designed to support both Portrait and Landscape orientations.

### Requirements
- Unity **2022.3 LTS** (any 2022.3.x)

### How to open
- Open the project root folder (`tic_tac_toe/`) in Unity Hub.

### Scenes
- `Assets/Scenes/Play.unity` (main menu)
- `Assets/Scenes/Game.unity` (gameplay)

### WebGL build
- Build target: **WebGL**
- Recommended: disable “Auto Graphics API” if you see WebGL issues, and use default compression settings initially.
